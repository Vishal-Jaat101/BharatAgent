## 🚀 What is BharatAgent?

BharatAgent is a voice-first, multilingual AI agent platform built for India.
Speak or type a task in any Indian language — Hindi, Tamil, Telugu, Bengali,
Marathi, Gujarati, Kannada, Malayalam, Punjabi or English — and BharatAgent
executes it using a network of intelligent background agents.

Think of it as **Jarvis for Bharat**.

---

## ✨ Features

- 🎤 **Voice Input/Output** — Speak tasks in 10+ Indian languages
- 🌐 **Multilingual AI** — Powered by Sarvam AI APIs
- 🧠 **Graph Memory** — Neo4j AuraDB knowledge graph remembers everything
- ⚡ **Background Workflows** — Render Workflows for durable agent execution
- 📱 **Mobile App** — Built with Expo (React Native)
- 🤖 **5 Agent Types** — General, Summarize, Translate, Remind, Search
- 📚 **History** — Full task history with language and agent type tracking
- 🔄 **Smart Suggestions** — Graph intelligence suggests related tasks

## 🛠️ Tech Stack

### Backend

- Node.js + Express.js
- Neo4j AuraDB (Graph Database)
- Sarvam AI APIs (STT, TTS, Translation)
- Render Workflows (Background Jobs)
- Helmet, Morgan, Rate Limiter

### Frontend

- React Native (Expo SDK 56)
- React Navigation (Bottom Tabs)
- Axios
- Expo Vector Icons
